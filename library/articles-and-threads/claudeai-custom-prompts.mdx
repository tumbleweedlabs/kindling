# ClaudeAI Custom Prompts

**A community collection of effective custom system prompts for Claude — real configurations that practitioners use for specific workflows.**

## What it covers

This community thread aggregates custom system prompts that Claude users have found effective for specific use cases. Rather than theoretical prompting advice, it's a practical exchange of configurations that people are actively using — covering areas like writing assistance, coding workflows, research tasks, and behavioral shaping (like reducing sycophancy, enforcing brevity, or maintaining a specific persona).

Custom system prompts are one of the most underused tools for getting better results from Claude. This thread is a shortcut: instead of discovering effective patterns yourself, you can review what's working for others and adapt it.

<Tip>
  **Read this when** you want to improve Claude's behavior for a specific workflow and would rather start from community-tested prompts than experiment from scratch.
</Tip>
